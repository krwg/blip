## ⚠️ Hotfix (если BLIP не запускается после установки 1.0.1)

Ошибка **`Cannot assign to read only property 'trustState'`** — скачайте **свежий** `BLIP-Setup-1.0.1.exe` с этого релиза (пересобран после исправления) или дождитесь **1.0.2**.

---

## ⚠️ Обновление с v1.0.0 (Windows)

**Автообновление с 1.0.0 не сработает** — это ограничение уже установленной версии, не баг релиза.

1. Скачайте **`BLIP-Setup-1.0.1.exe`** в **Assets** ниже.
2. Запустите установщик **один раз** (можно поверх BLIP 1.0.0).
3. Настройки, чаты и конфиг сохраняются.

С **v1.0.1** следующие патчи (1.0.2 и т.д.) снова можно получать через **Настройки → Обновления**.

---

## ⚠️ Upgrading from v1.0.0 (Windows)

**Auto-update from 1.0.0 will not install this release.** Download **`BLIP-Setup-1.0.1.exe`** from **Assets** and run it once (over your current install). From **v1.0.1** onward, later versions can update from the app again.
