Ты в сети. Ты сигнал.

Десктоп-мессенджер для локальной сети: чат и звонки без облака и без серверов. UDP + TCP discovery, WebRTC. Electron + Vite, BLIP ID 1–64.

Архитектура · Architecture

P2P LAN

Нет аккаунтов в облаке: узнаём соседей по UDP broadcast и mDNS, сообщения по TCP между пирами.

Звонки

WebRTC: голос/видео в отдельном окне, сигналинг по тому же TCP, что и чат.

Стек

Electron 35, Vite 6, vanilla JS. Windows — основная цель сборки NSIS/portable.
