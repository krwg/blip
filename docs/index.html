Десктоп-мессенджер для LAN / Hamachi / Radmin VPN: чат, группы, голосовые каналы, звонки 1:1 и групповые — без облака. UDP + TCP + WebRTC, BLIP ID 1–64.

Star on GitHub Releases

Статическая витрина GitHub Pages в акценте «Сланец» (#94a3b8), как в приложении. Сборка — Electron; разработка — CONTRIBUTING в репозитории.

Архитектура →

P2P LAN

UDP broadcast + mDNS, TCP между пирами, Mesh Handshake (Ed25519). Без регистрации и без серверов.

Звонки

Окна 1:1 и группового звонка, голосовые каналы в группах, демонстрация экрана 720p+, chiptune СИГНАЛ / ПУЛЬС.

Чат

Цитаты, медиа-просмотр и YouTube в чате, реакции, Ctrl+F, файлы до 100 ГБ, буфер обмена по LAN.

Облик

Свет / тьма / авто, 16 акцентов (Сланец), арт-обои, своё фото профиля по LAN, пиксельные переключатели.

Mesh Pulse

Пинг раз в минуту, задержка под абонентом, статус-текст в LAN, избранные и блокировка в настройках.

Стек

Electron 35, Vite 6, vanilla JS/CSS. Windows — NSIS и portable; полный интерфейс EN / RU.
