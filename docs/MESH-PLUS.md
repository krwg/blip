# BLIP · MESH+

**MESH+** is an optional tier on top of BLIP. **FREE** includes the full LAN messenger: chat, calls, groups (beta), Mesh Pulse, avatars, themes, and core sounds.

Activate in **Settings → MESH+** with your `BLIP-XXXX-…` key.

## About the subscription

MESH+ was introduced as a **test** — a way to gain experience building a **license-checked desktop app** (offline Ed25519 activation, author verification, tier gates in main and renderer).

| | |
|---|---|
| **Who issues keys** | The author personally — not a store or automated billing |
| **How to get MESH+** | Email **[blipteam@icloud.com](mailto:blipteam@icloud.com)** — keys are issued **free of charge** |
| **Why it exists now** | Early access to **new features** — subscribers test them first on the LAN |
| **Long term** | Everything that ships under MESH+ today is planned to become **FREE for everyone** later |
| **What stays special** | Broader **customization** (themes, sounds, icons, status GIF, Signal Corps tools) and the **MESH+ badge** on your profile for peers |

MESH+ is not paywalled core messaging: chat, voice/video, discovery, and Mesh Pulse stay FREE. The tier is a lab for licensing UX and a thank-you channel for testers who help ship features safely.

## Settings

- **MESH+** — activation and benefits overview.
- **Network** — session stats; optional Signal Corps clipboard mesh sync.
- **Achievements** — optional, off by default.
